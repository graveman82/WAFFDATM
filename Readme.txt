WAFFDATM - библиотека для быстрого создания демок, приложений тестирования или анализа или быстрой разработки прототипов для учебных примеров, предназначенных для запуска в ОС семейства Windows.

Можно опрашивать состояния клавиш, получать дельты смещений курсора.
Для генерации справки по библиотеке необходимо использовать doxygen.
